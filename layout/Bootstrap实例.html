Chrome

Google Chrome ,又称谷歌浏览器,是一个由谷歌公司开发的网页浏览器.

Firefox

Mozilla Firefox，中文俗称“火狐”（正式缩写为Fx或fx），是一个由Mozilla开发的自由及开放源代码的网页浏览器.

Safari

Safari是一款由苹果开发的网页浏览器，是各类苹果设备（如Mac、iPhone、iPad、iPod Touch）的默认浏览器。Safari使用WebKit浏览器引擎.

IE

Internet Explorer（旧称Microsoft Internet Explorer和Windows Internet Explorer，简称IE或MSIE），是微软所开发的图形用户界面网页浏览器。自从1995年开始，内置在各个新版本的Windows操作系统，也是微软Windows操作系统的一个组成部分。
